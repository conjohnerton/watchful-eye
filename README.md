## quickstart
### Run liqui-bot's client 
  1. `yarn install`
  2. `yarn start`

### Run the local chain
> in a second terminal window:
  1. `yarn chain`

### Compile and deploy contracts
> in a third terminal window:
  1. `yarn deploy`
